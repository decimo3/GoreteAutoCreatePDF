# PDF_BOT

Programa desenvolvido para criar PDFs de pastas contendo imagens (somente imagens).

O programa abre uma janela de diálogo solicitando uma pasta. O caminho selecionado será usado como base para procurar as pastas com imagem para gerar os PDFs.

Os PDFs serão gerados com o nome da pasta a qual as imagens estão contidas. A qualidade das imagens será reduzida para caber em um e-mail.
